# Pulse Rubroek — design iterations

Four working visual and layout directions for the Pulse Rubroek website.
The original site is bundled in `source/`, so this repository can be hosted as
a standalone static website.

## Live site

GitHub Pages deploys the `main` branch to:

<https://maxi-taxi01.github.io/Pulse-variattions/>

## Open locally

Because each iteration loads the bundled site in a same-origin frame and then
adds its own design layer, run the repository through a local web server:

```powershell
py -m http.server 8765 --bind 127.0.0.1
```

Then open <http://127.0.0.1:8765/>.

## Directions

- **Civic Editorial** — calm, credible and publication-led.
- **Pulse After Dark** — energetic, event-like and expressive.
- **Neighbourhood Notebook** — warm, human and approachable.
- **Systems Atlas** — compact, analytical and map-first.

All original interactions and embedded maps remain available.
